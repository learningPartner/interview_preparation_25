What is Hoisting In JavaScript

Hoisting is a JavaScript behavior where in Memory Allocation phase variable which are declared with var are assigned undefined value and Function declation stored directly, so Before initialization we can acces variable and we can invoke function also.

  • Variable declared with var and Normal function supports hoisting
  • let & const does't support hoisting as it does with var They will be tempral Dead zone till it initialize
  • Normal function only support Hoisting not arrow Function
  • class also does't support Hoisting
 console.log(name); // ✅ Undefined
 var name = "John";
 console.log(name); // John
        
var name;  // Hoisted to the top
console.log(name); // undefined
name = "John";
console.log(name); // John
